Definition des tables :
- Table auteurs
( nomAuteur, prenomAuteur, date de naissance, date de mort, nationalité, mouvement, biographie)

- Table oeuvre
( #nomAuteur, #prenomAuteur, #nomMusee, nom, date de création, date d'exposition, info)

- Table Musée
( nomMusee, Ville )

- Table commentaire
( id, contenu, #nom, date)

Classes nécessaires : 

- Auteur ( liste d'attribut )
- Oeuvre ( liste d'attribut, Auteur, Musee )
- Musee ( Liste d'attribut )
- commentraire ( liste d'attribut, oeuvre )
